JSON Data API

Attributes

Query Params

  • name: Person's full name (e.g. "name=Mike Smith").
  • race: Person's race (e.g. "race=Caucasian-American/White").
  • sex: Person's physical sex (e.g. "sex=male").
  • cause: Cause of person's death (e.g. "cause=gunshot").
  • state: Two letter abbreviation for American state (e.g. "state=ca").

Pagination

Query Params

  • limit: How many entries to show (e.g. "limit=10"). Setting limit to false returns all entries.
  • page: Which page of entries to show (e.g. if limit is 10, then page 2 returns recoreds 11-20).
